I'm a Lead Pipeline Technical Director at Sun Creature in Copenhagen. My job, in plain terms: I figure out what's slowing people down, then I build the tools to fix it. I've been doing this for 7+ years across animation, film, and games.

I've built integrations across 10+ DCCs and services — Maya, Adobe CC, ToonBoom, TVPaint, Fusion, Flow (ShotGrid), and plenty more. If artists are fighting their tools instead of making great work, that's where I step in.

I care a lot about doing right by the people I work with. I document things properly, I communicate early, and I make sure nobody gets stuck waiting on me. I've been lucky enough to ship work for Netflix, Blizzard, Disney, Riot Games, Supercell, EA, Lego, and others.

Pipeline Tools

import pipeline_tools

A collection of standalone Python tools I built for animation and VFX production. Each one solves a real problem I've encountered on the job. All open source, all built with PySide2/PyQt5.

✔
Asset Validator
Checks naming conventions, folder structure, and file metadata against configurable rulesets. Catches problems before they reach downstream.
PythonPySide2YAMLRegex
📦
Scene Publisher
Lightweight publish system with automatic versioning, thumbnail generation, status tracking, and configurable targets.
PythonPySide2JSONVersioning
✏
Batch Renamer
Rename and organize production files with regex, presets, find-and-replace, and numbering — with live preview before applying.
PythonPySide2RegexPresets
📊
Production Tracker
Shot and asset status dashboard with filtering, artist assignment, progress stats, and status color coding.
PythonPySide2JSONDashboard
🚀
DCC Launcher
Launch DCCs with the right environment per project. Manages software versions, env vars, and configs in one place.
PythonPySide2YAMLEnv Vars
⚙
Pipeline Config Manager
Visual editor for YAML/JSON pipeline configs. Supports hierarchical settings, environment overrides, and schema validation.
PythonPySide2YAMLJSON
🔄
Media Converter
Batch convert between image and video formats. Handles EXR, PNG, JPG, TIFF, MOV, MP4 — the formats you actually use in production.
PythonPySide2PillowFFmpeg
🎬
Frames ↔ Video
Convert image sequences to video and back. Supports EXR, PNG, JPG frames and MOV, MP4 outputs with frame range control.
PythonPySide2FFmpegSequences
🔗
Video Concatenator
Join multiple videos with drag-and-drop reordering. Toggle audio on/off per clip and choose output format.
PythonPySide2FFmpegAudio
📤
File Collector
Gather files from anywhere, package them for delivery with manifests, checksums, and folder structure validation.
PythonPySide2ManifestDelivery
📥
File Ingestor
Ingest incoming files into structured pipeline folders. Validates, renames, organizes, and registers assets automatically.
PythonPySide2YAMLAutomation
